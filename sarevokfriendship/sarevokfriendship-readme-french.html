Introduction


Un élément dont beaucoup trouvent les PNJ de Bioware dépourvus, c'est la relation plus complexe permise par les mods PNJ modernes - surtout en ce qui concerne les "amitiés", les relations que les PNJ ont avec le PJ lorsqu'ils n'ont pas une romance active.

Ce mod ajoute une série de dialogues à Sarevok, le frère du PJ, et développe ses expériences et ses difficultés.

Visitez le site web ou le forum pour les dernières mises à jour.

Installation


Windows

Sarevok Friendship est un mod WeiDU, compressé via une archive extractable .ZIP. Décompressez simplement l'archive dans votre répertoire Baldur's Gate II et exécutez " setup-sarevokfriendship.exe ".

Pour désinstaller, exécutez à nouveau " setup-sarevokfriendship.exe " et sélectionnez l'option de désinstallation.



Compatibility and Installation Order

Le mod Sarevok Friendship devrait être compatible avec n'importe quel mod WeiDU, nous ne pouvons cependant pas tous les tester. Il n'est compatible qu'avec BGII : ToB. Si vous rencontrez une erreur, veuillez contactez K'aeloree ou bien postez dans les forums.

Bien qu'il ne soit pas requis pour faire correctement tourner Sarevok Friendship, il est toujours utile d'avoir installé la dernière version du BG2 Fixpack.

Foire Aux Questions


Q : Sarevok Friendship est-il compatible avec le mod XYZ ?
A : Il devrait l'être oui, mais nous n'avons pas testé tous les mods disponibles. Si vous rencontrez une erreur, veuillez contactez K'aeloree ou bien postez dans les forums.

Q : Sarevok Friendship est-il compatible avec Sarevok Romance ?
A : Oui ; les conversations d'amitié s'arrêteront une fois que la romance débutera.

Q : Quand débutera l'amitié ?
A : Elle devrait débuter peu de temps après qu'Sarevok ait rejoint le groupe

Q : Ce mod ne respecte pas *du tout* le personnage
A : Je suis toujours à la recherche de moyens d'améliorer l'incarnation des personnages dans ce mod, envoyez-moi toute suggestion ou critique (kae@spellholdstudios.net) ou bien postez sur les forums.

Q : As-tu l'intention d'étendre le mod en lui ajoutant d'autres dialogues dans le futur ?
A : Non, il n'est pour l'instant pas prévu de faire des ajouts au mod, mais si vous avez des suggestions, n'hésitez pas !

Q : Combien de dialogues ont été ajoutés en tout ?
A : Il y a un total de 7 nouveaux dialogues ajoutés par le mod. Certains d'entre eux se produisent sous certaines conditions - pour plus d'informations, n'hésitez pas à demander sur le forum.

Q : Ce mod change-t-il le jeu d'une façon ou d'une autre, hormis les nouveaux dialogues de Sarevok ?
A : Normalement non.

Q : Y a-t-il une chance pour que des dialogues initiés par le joueur soient ajoutés dans le futur ?
A : Ce n'est pas prévu pour l'heure, mais si des joueurs sont intéressés je serais ravie d'envisager d'en ajouter lors des versions futures.

Q : Je souhaiterais traduire Sarevok Friendship dans une autre langue. Pourrais-tu TRAifier le mod ?
A : Bien sûr, j'en serai heureuse. Si vous souhaitez traduire, envoyez-moi un mot et je verrai ce que je peux faire pour la prochaine version.
